# JS_question

This folder contains simple Node.js practice scripts for common algorithm and array/string tasks.

## Files

- `fibonacci.js` - generates a Fibonacci series up to a given index and prints the result.
- `flatten.js` - flattens a nested array using `Array.flat(Infinity)` and prints the output.
- `palindrome.js` - checks whether a string is a palindrome and prints the original string along with the boolean result.
- `reverse.js` - reverses a string and prints the reversed value.
- `removeDuplicate.js` - removes duplicate values from an array using a `Set` and prints the deduplicated array.
- `test.js` - demonstrates common array methods such as `filter`, `map`, `reduce`, `find`, `some`, `every`, `includes`, `sort`, `join`, `slice`, `splice`, `concat`, `reverse`, `forEach`, `findIndex`, `indexOf`, `lastIndexOf`, `push`, `pop`, `shift`, `unshift`, and `Array.isArray`.

## Subfolders

### `Rectangle/`

Contains rectangle pattern printing exercises, such as printing repeated rows of numbers.

### `Triangle/`

Contains triangle pattern printing exercises, such as printing growing rows of stars.

## Usage

Run any script with Node.js from the project root directory:

```bash
node fibonacci.js
node flatten.js
node palindrome.js
node reverse.js
node removeDuplicate.js
node test.js
```

For subfolder scripts, run:

```bash
node Rectangle/rec1.js
node Triangle/tri1.js
```

## Notes

- This repository is designed for learning and practicing basic JavaScript concepts.
- Most scripts print results directly to the console.
